# python-rapido-
🚲 Python Rapido – Ride Booking System

A simple Python console-based ride booking application inspired by Rapido.
This project allows users to view available ride types, book rides based on distance, view booking details, cancel rides, and confirm rides with automatic fare calculation and discount.

📌 Features

🚗 View available ride types (Bike, Auto, Cab)

📏 Enter ride distance in kilometers

💰 Automatic fare calculation

🎟 10% discount on rides above ₹200

📋 View current booking details

❌ Cancel booked rides

🧾 Generate final ride bill

📱 Simple interactive menu system

🛠 Technologies Used

Python

CLI (Command Line Interface)
